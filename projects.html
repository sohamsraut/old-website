The Evil Typing Test

The Evil Typing Test is, well, an evil typing test implemented as a web application. The main idea behind it was that text in normal typing tests makes sense, so what if there was a typing test in which the text didn't make sense? Thus, the Evil Typing Test was born, with random text retrieved from the FakerAPI.

GameZone

GameZone is a quick, responsive text-based game arcade consisting of 5 games: Mastermind, Tic-Tac-Toe, Hangman, Rock-Paper-Scissors, and Quizzer. It was created as part of a school project by my team. It is implemented in Python, and uses a CSV-based login and records system to store user data and records for each game.
